# Coastal Forcing Directory
This directory contains necessary tools to create forcing input files for coastal models including SCHISM and SFINCS

The 'atmo' directory has the script to create the forcing input file for SFINCS
The 'tidal' dirctory has scripts to create boundary tidal data for SCHISM and SFINCS. 
The 'calib' dirctory has scripts to run SCHISM calibration workflow using NWM retrospective data and STOFS or TPXO data. 
The 'Observed_Waterlevel_Scripts' dirctory has scripts to download the observed water level data from NOAA. 
The 'FVCOM_Download' dirctory has scripts to download the FVCOM datga for the Great Lakes from NOAA. 
The 'compare_observed' dirctory has scripts to compare observed water levels with simulated water levels.
